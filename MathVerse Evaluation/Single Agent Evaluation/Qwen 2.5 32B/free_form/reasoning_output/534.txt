### Problem Analysis:

- Maximum point: $\left( \frac{1}{4}\pi, -2.2 \right)$
- Minimum point: $\left( \frac{3}{4}\pi, -8.5 \right)$

#### Step 1: Understanding the Amplitude
The amplitude of a trigonometric function is half the distance between its maximum and minimum values. Mathematically, it is calculated as:
\[
\text{Amplitude} = \frac{\text{Maximum value} - \text{Minimum value}}{2}
\]

From the graph:
- The maximum value of the function is $-2.2$.
- The minimum value of the function is $-8.5$.

#### Step 2: Calculate the Distance Between Maximum and Minimum Values
The difference between the maximum and minimum values is:
\[
\text{Difference} = \text{Maximum value} - \text{Minimum value} = -2.2 - (-8.5) = -2.2 + 8.5 = 6.3
\]

#### Step 3: Calculate the Amplitude
The amplitude is half of this difference:
\[
\text{Amplitude} = \frac{\text{Difference}}{2} = \frac{6.3}{2} = 3.15
\]
